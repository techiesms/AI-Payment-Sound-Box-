Installation Instructions

  1. Use Google Chrome, Microsoft Edge, Brave, or Opera.
  2. Connect your ESP32-S3 board using a USB data cable.
  3. Close Arduino IDE Serial Monitor if open.
  4. Click the Flash Firmware button.
  5. Select the correct serial port.
  6. Wait for firmware upload to complete.
  7. The device will restart automatically.

Troubleshooting

  • Use a USB data cable, not a charging cable.
  • Install CP210x or CH340 USB drivers.
  • Close all serial monitor applications.
  • Try another USB port.
  • Hold BOOT while connecting if flashing fails.
  • Refresh the page and try again.
  • Use the latest version of Chrome or Edge.
